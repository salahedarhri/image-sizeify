    git clone https://github.com/salahedarhri/image-sizeify.git

2. Navigate to the project directory & Install:
    ```bash
    cd image-sizeify
    composer install
    npm install
    ```
    
4. Copy .env.example and fill it with your own infos:
    ```bash
    cp .env.example .env
    php artisan key:generate
    ```

5. Migrate:
    ```bash
    php artisan migrate
    ```
